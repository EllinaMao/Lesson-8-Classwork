# Main Task
Симулятор живой фермы (Farm Simulator) Создайте классовый или функциональный компонент FarmSimulator, который:

1) Хранит массив животных в state. Каждое животное имеет поля: id, type (например, корова, курица, свинья), name, age, hunger (число от 0 до 100).
2) Позволяет добавлять новое животное с выбором типа, имени и возраста.
3) Отображает все животные в виде карточек с их данными и шкалой голода (например, полоской прогресса).
4) Позволяет кормить животное, уменьшая его голод на определённое значение, и старить животных (увеличение возраста и голода при обновлении времени).
5) При каждом обновлении состояния (кормление или старение) сохраняет данные в localStorage.
6) Позволяет удалять животное с фермы.


# Extras 
1) Создайте React-компонент, который позволяет пользователю переключаться между светлой и темной темами. При переключении тема должна изменяться, и компонент должен отобразить текущую тему.
2) 	Создайте React-компонент, который будет функционировать как простой калькулятор. Компонент должен иметь кнопки для ввода чисел и операций (+, -, *, /). При вводе выражения и нажатии кнопки "Рассчитать", компонент должен вычислить результат и отобразить его.
 	
3) Создайте React-компонент для формы регистрации пользователя. Форма должна содержать поля для ввода имени, email и пароля. После заполнения формы и нажатия кнопки "Зарегистрироваться", компонент должен проверять введенные данные (например, проверять формат email и длину пароля) и выводить сообщения об ошибках, если данные введены неверно.

4) Разработайте компонент «Калькулятор чаевых». Приложение должно позволять пользователю вводить общую сумму счета, выбирать процент чаевых и отображать рассчитанную сумму чаевых и общую сумму (сумма счета + чаевые).

5) Динамическая форма с несколькими полями. Создайте компонент DynamicForm, который:
    1) Хранит состояние в виде объекта с полями формы (name, email, phone и т.д.).
    2) Позволяет пользователю динамически добавлять новые поля (например, "Дополнительная информация").
    3) Каждое поле управляется через state и может быть изменено пользователем.
    4) Показывает текущее состояние формы в реальном времени под формой.
    5) Добавьте кнопку "Очистить форму", которая сбрасывает все поля.
